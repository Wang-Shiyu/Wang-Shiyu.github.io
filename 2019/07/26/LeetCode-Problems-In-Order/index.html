2 Add Two Numbers

C++

Kind of ugly, but it works : ) Time:16ms

Note:

  • create a new listed link, do not modify the parameters(l1, l2)

  • Initialize current node to dummy head of the returning list.

  • create new node to the head of l1, l2 respectively.

  • Keep track of the carry using a variable, initialize carry to 0.

  • Use % and / to make the code concise. Like updating $$sum = l1->val + l2->val +carry$$

  • Update carry by $$sum/10$$

  • Use the value $$ sum%10 $$ when creating a new node.

java demo

Time Complexity:$$O(max(m,n))$$

Space Complexity:$$O(max(m,n)) $$
